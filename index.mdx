## What is Airport Manager?

Airport Manager is a Django web application that wraps the [AirportGap API](https://airportgap.com) in a clean, browser-based interface. It lets you:

- **Search** any airport worldwide using its IATA (e.g. `MEX`) or ICAO (e.g. `KJFK`) code
- **Save favorites** — bookmark airports and attach personal notes to them
- **Calculate distances** — get the distance in kilometers, miles, and nautical miles between any two airports
- **Compare airports** — view two airports side-by-side to compare their details

Authentication with the AirportGap API is handled via a bearer token stored in your environment, so your favorites are tied to your own account.

## Get started

<Steps>
  <Step title="Clone the repository">
    Download the source code from GitHub.
  </Step>
  <Step title="Set your API token">
    Create a `.env` file with your AirportGap bearer token.
  </Step>
  <Step title="Run the server">
